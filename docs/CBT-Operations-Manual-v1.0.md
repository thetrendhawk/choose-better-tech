# Choose Better Tech Operations Manual v1.0

Version: 1.0  
Status: Active  
Purpose: Governing operating manual for the Choose Better Tech business.

## Mission

Choose Better Tech exists to help beginners make confident software purchasing decisions through transparent, evidence-based editorial content.

Revenue is generated through affiliate partnerships, but recommendations are driven by reader fit rather than commission potential.

## Vision

Build the internet's most trusted beginner-focused software recommendation brand.

Success is measured by trust first, revenue second.

## Core Principles

### Readers First

Every article should help someone make a better decision.

If affiliate links disappeared tomorrow, the article should still be worth reading.

### Evidence Before Opinion

Recommendations begin with research, never assumptions.

Every important conclusion must be supported by evidence.

### Context Matters

There is no universally best software.

Every recommendation is conditional.

Always answer: best for whom?

### Transparency

Disclose affiliate relationships.

Discuss tradeoffs.

Explain limitations.

Never hide weaknesses.

### Long-Term Trust

Choose Better Tech optimizes for returning readers, reputation, and authority rather than quick conversions.

## Publication Pipeline

Every article follows this workflow:

1. Opportunity Score
2. Perplexity Deep Research
3. Evidence Review
4. CBTEP
5. Editorial Outline
6. Codex Implementation
7. Publisher Readiness Review
8. Publish
9. Internal Linking
10. Index Monitoring
11. Performance Review

No exceptions.

## Decision Rule

Before starting any task, ask:

Will this increase the probability of earning the next affiliate commission?

If the answer is no, the task is postponed or discarded.
