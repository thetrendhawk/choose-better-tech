# Publisher Readiness Checklist

Use this checklist before publishing any new article.

## Editorial

- [ ] The article answers the reader's decision question.
- [ ] The recommendation is contextual.
- [ ] Tradeoffs are clearly explained.
- [ ] Weaknesses are not hidden.
- [ ] Affiliate disclosure is visible.
- [ ] The article avoids unsupported superlatives.

## Evidence

- [ ] Official documentation reviewed.
- [ ] Independent reviews reviewed.
- [ ] Community feedback reviewed.
- [ ] Pricing verified.
- [ ] Alternatives considered.
- [ ] Evidence gaps identified.

## SEO

- [ ] Primary keyword is clear.
- [ ] Search intent is satisfied.
- [ ] Meta title is written.
- [ ] Meta description is written.
- [ ] Headings are semantic.
- [ ] FAQ section included when appropriate.
- [ ] Internal links added.

## UX

- [ ] Page is easy to scan.
- [ ] CTAs are placed after useful information.
- [ ] Tables are readable on mobile.
- [ ] Disclosure is visible.
- [ ] Related articles are included.
- [ ] No broken links.

## Technical

- [ ] npm run build passes.
- [ ] No TypeScript errors.
- [ ] Route works.
- [ ] Page is responsive.
- [ ] Page is linked from relevant pages.
- [ ] Sitemap is updated if applicable.

## Publishing Standard

Target Publisher Readiness Score: 9.0 or higher.
