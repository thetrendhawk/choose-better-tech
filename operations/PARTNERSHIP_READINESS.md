# Partnership Readiness

Version: 1.0  
Status: Active  
Owner: Choose Better Tech  
Last Updated: 2026-07-02

## Purpose

This checklist determines whether Choose Better Tech is ready to apply for a new affiliate or partnership program.

## Website

- [ ] Website is publicly accessible.
- [ ] Core navigation works.
- [ ] Contact page is available.
- [ ] About page clearly explains the business.
- [ ] Affiliate disclosure page is available.
- [ ] Privacy policy is available.
- [ ] Terms page is available.
- [ ] Mobile layout is usable.

## Content

- [ ] Relevant category content is published.
- [ ] At least one strong commercial-intent article supports the application.
- [ ] Articles include original decision guidance.
- [ ] Articles disclose affiliate relationships where relevant.
- [ ] Internal links connect related articles.
- [ ] Content is current enough for partner review.

## Traffic

- [ ] Google Search Console data reviewed.
- [ ] Google Analytics data reviewed.
- [ ] Bing Webmaster Tools data reviewed.
- [ ] Traffic baseline recorded.
- [ ] Top pages identified.
- [ ] Any major traffic gaps noted.

## Legal

- [ ] Affiliate disclosure is visible.
- [ ] Privacy policy is reviewed.
- [ ] Terms page is reviewed.
- [ ] Sponsored and nofollow link practices are understood.
- [ ] Program-specific disclosure requirements are reviewed.

## Business

- [ ] Business name is consistent across application materials.
- [ ] Website URL is correct.
- [ ] Contact email is current.
- [ ] Business description is ready.
- [ ] Payment and tax information can be supplied when requested.

## Analytics

- [ ] Google Analytics access is available.
- [ ] Search Console access is available.
- [ ] Bing Webmaster Tools access is available.
- [ ] KPI dashboard has the latest available numbers.
- [ ] Application notes include relevant performance context.

## Social

- [ ] Pinterest presence reviewed.
- [ ] Instagram presence reviewed.
- [ ] LinkedIn presence reviewed.
- [ ] Social links are accurate if used in applications.
- [ ] Distribution plan exists for partner-relevant articles.

## Editorial

- [ ] Product coverage is balanced.
- [ ] Pros and cons are included.
- [ ] Alternatives are considered.
- [ ] Claims are supported by evidence.
- [ ] Recommendations are contextual.
- [ ] Weaknesses are not hidden.

## Reapplication Rules

- Wait at least 30 days before reapplying unless the program states a different policy.
- Document the rejection reason before reapplying.
- Improve the weakest readiness area before submitting again.
- Update the application log after each application or reapplication.
- Keep partner communications professional and concise.
